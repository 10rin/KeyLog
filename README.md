# KeyLog (打鍵音量計測・分析アプリケーション)

キーボードを叩いたときの打鍵音（マイク入力）をリアルタイムで監視・測定し、キーごとの打鍵音量を記録・分析するmacOS向けのSwiftUIアプリケーションです。
特に「Enterキーを他のキーよりも強打しているか」を自動判定し、実験データをテキストファイルとして書き出すことができます。

---

## 🚀 主な機能

1. **リアルタイム音量監視**
   * マイクからの音声入力を `AVAudioEngine` を用いてリアルタイムに解析し、音量の指標（RMS値）を計算します。
2. **グローバルキー入力検知**
   * アプリがバックグラウンドにある状態でも、キーボードのどのキーが押されたかを判定します。
3. **Enterキー強打の分析**
   * Enterキー（Returnキー）とそれ以外のキーの平均音量を比較し、Enterキーが強く叩かれているかを判定します。
4. **全キーの統計レポート**
   * 実験中に押されたすべてのキーの「打鍵回数」と「平均音量」の詳細リストを生成します。
5. **ファイル保存機能**
   * 分析結果を任意の場所にテキストファイル（`.txt`）として保存できます。

---

## 🛠 動作環境

* **OS**: macOS 14.0 (Sonoma) 以上
* **開発・ビルド環境**: Swift 5.9 以上 / Xcode 15 以上
* **必要な権限**: 
  * **マイク入力の許可**（音量測定のため）
  * **アクセシビリティ（ユーザー補助）の許可**（バックグラウンドでのキー入力を検知するため）

---

## 📦 起動・実行方法

### 方法 1: ターミナルから実行する場合
プロジェクトのルートディレクトリ（`Package.swift` がある場所）で、以下のコマンドを実行します。

```bash
swift run
```

### 方法 2: Xcode から実行する場合
1. `Package.swift` を Xcode で開きます。
2. ターゲットに `KeyLog` (My Mac) を選択します。
3. `Product` -> `Run` (または `Cmd + R`) を押して実行します。

---

## ⚠️ 初回起動時の重要設定（重要）

このアプリはシステム全体のキー入力とマイク入力を監視するため、macOSのセキュリティ設定で**権限の許可**が必要です。

1. **マイクの許可**
   * アプリ起動時にマイクへのアクセス要求ダイアログが表示されたら「OK」を選択してください。
2. **アクセシビリティの許可（キー入力監視用）**
   * アプリを起動して「Start」を押してもキー入力が反応しない場合、以下の設定を行ってください。
   * macOSの **「システム設定」 ＞ 「プライバシーとセキュリティ」 ＞ 「アクセシビリティ」** を開きます。
   * リスト内の **「ターミナル」** (または実行しているアプリ名) のスイッチを **オン** にしてください。
